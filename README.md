# SovereignKeep

This repo is for self-hosting only. If you'd like SovereignKeep to handle all of this for you, including maximum legal protection, security updates, and hosting, go to https://www.SovereignKeep.com.

### Requirements

- [Docker](https://www.docker.com/community-edition#/download)
- [Docker Compose](https://docs.docker.com/compose/install/) (already included with some Docker installations)
- For maximum legal protections, you'll want to be a company registered in Wyoming

*These dependencies are free to use.*

### Running
`docker-compose up`
